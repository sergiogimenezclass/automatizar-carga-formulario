Automatizar un Google Form de terceros

Problema real: tenemos que cargar periódicamente una bitácora de desarrollo, pero los datos ya existen en Excel o CSV.

Python Pandas Playwright Google Forms
El problema

Cargar a mano es repetitivo, lento y propenso a errores

PeriodicidadEl formulario puede ser diario, semanal o mensual.
Datos repetidosMuchas respuestas salen de una planilla ya preparada.
Cuenta GoogleDebe enviarse desde el usuario correcto, ya autenticado.
Caso concreto

Formulario: Bitácora de Estado de Desarrollo de Software

Campos detectadosEstado, fecha del reporte, detalle técnico y módulo/componente trabajado.
Tipos de entradaOpciones, fecha, texto libre y desplegable.

Ejemplos de opciones: Backend, Base de Datos, UI/UX e Integración de APIs.

Solución elegida

Un navegador controlado por código

Playwright abre Chrome/Chromium, entra al formulario, completa cada campo y presiona “Enviar”, como si fuera una persona.

Automatización

Completar el formulario con Playwright

from playwright.sync_api import sync_playwright

FORM_URL = "https://docs.google.com/forms/..."

with sync_playwright() as p:
    browser = p.chromium.launch_persistent_context(
        user_data_dir="./perfil_google",
        headless=False
    )

    page = browser.new_page()
    page.goto(FORM_URL)

    page.get_by_text("Funcional/Terminado").click()
    page.get_by_label("Fecha del reporte").fill("18/06/2026")
    page.get_by_label("Detalle técnico").fill("Se corrigió el login.")
    page.get_by_text("Escolher").click()
    page.get_by_text("Backend").click()

    page.get_by_text("Enviar").click()
